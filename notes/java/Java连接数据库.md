# Java 连接 MySQL 8.0 数据库

## 1. 准备

### 1.1 安装 *Maven* 的驱动宝

首先在 *pom.xml* 的文件中添加如下内容。

``` xml
    <dependencies>

        <!-- 导入 MySQL 的Maven依赖 -->
        <dependency>
            <groupId>mysql</groupId>
            <artifactId>mysql-connector-java</artifactId>
            <version>8.0.13</version>
        </dependency>

    </dependencies>
```
